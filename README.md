# Dashboard-de-Credito

Neste projeto, desenvolvi uma aplicação em R Shiny para análise do mercado de crédito brasileiro com base em dados do Banco Central do Brasil (SGS). O código foi estruturado para carregar dados pré-processados, mapear séries econômicas por grupo, carteira, segmento, categoria e subcategoria, e transformar essas informações em um dashboard interativo com foco analítico e gerencial.

Principais funcionalidades do projeto
carregamento e organização de séries de crédito do Banco Central;
estruturação de indicadores como saldo, concessões, juros e inadimplência;
segmentação por crédito livre e direcionado, PF e PJ;
criação de indicadores consolidados e comparativos;
cálculo de variações m/m e a/a;
exibição de KPIs e resumos gerenciais;
gráficos interativos com Plotly;
filtros dinâmicos por período e tipo de série;
comparação entre indicadores em nível, índice e variações;
interface dividida em abas para diferentes tipos de análise
